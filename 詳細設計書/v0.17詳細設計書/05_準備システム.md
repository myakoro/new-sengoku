# v0.17 詳細設計書：05_準備と追撃システム（盗賊討伐特化）

## 0. 本資料の目的

- 抽象的な「準備・実行」の連打抑制ではなく、具体的な戦術的トレードオフを設計する
- 盗賊討伐における「戦闘前の準備」と「戦闘終了後の追撃・討ち取り」のロジックを定義する
- 状況変化ロジック（放置や偏りへの反作用）を整理する

---

## 1. 準備システムの刷新

### 1-1. 設計変更の概要

- **小評定レベルの準備（廃止）**: 従来設計の「準備アクションを実行して倍率（0.6〜1.4倍）を得る」仕組みを廃止。理由はアプローチ選択（愚直/並/豪華）と役割が重複するため。
- **戦闘レベルの準備（新規）**: 盗賊討伐の下知（2週間、最大24ターン）の中で実行される具体的な戦術アクション。

### 1-2. 戦闘ターン内の準備アクション（交戦前）

敵と「攻撃開始」で接敵する前に、配下を活用して並列に準備を進める。

- **並列実行**: 主人公および各若党にアクションを1つずつ割り当て可能。
- **ターン進行**: 割り当て後、「ターンを進める」ことで時間が経過し、所要ターンに達すると結果が出る。
- **能力依存**: 成功率は担当者の「知略」などの能力値に依存する。
- **スキルツリー形式**: アクション選択UIは依存関係を示すツリー構造（[詳細偵察＆地形調査] → [夜襲/火計/作戦]等）で構成される。

| アクション | 所要T | 主な能力 | 効果 |
|-----------|:---:|:---:|------|
| **詳細偵察** | 3 | 知略 | 敵の正確な数・配置が判明。**工作・計画の共通起点** |
| **地形調査** | 4 | 政務 | 敵の退路を遮断。追撃効率+25%。**前提条件** |
| **夜襲計画** | 2 | 知略 | 解禁条件：詳細偵察＆地形調査完了。成功率の参照先を決定 |
| **夜襲準備** | 4 | 政務 | 解禁条件：計画完了。夜襲攻撃を選択可能にする |
| **火計計画** | 2 | 知略 | 解禁条件：詳細偵察＆地形調査完了。成功率の参照先を決定 |
| **火計準備** | 3 | 政務 | 解禁条件：火計計画完了。敵戦闘力-10%（開始5T） |
| **全体作戦計画** | 2 | 知略 | 解禁条件：詳細偵察＆地形調査完了。自軍戦闘力+10%（開始5T） |
| **戦闘前鼓舞演説**| 1 | 統率 | **攻撃開始の直前のみ・1回限定**。士気+5、攻撃力+5（開始3T） |
| **住民聞き込み**| 2 | 知略 | ターゲットの生活習慣を把握。初撃ダメージ+20% |
| **応援要請** | 4 | 統率 | 臨時足軽の追加（兵力増量） |
| **攻撃開始** | 0 | - | 「通常攻撃」または「夜襲攻撃」を選択 |

## 2. 実交戦中の特殊アクションとアタック分岐

### 2-1. アタックタイプの分岐と特殊判定
準備フェーズから「攻撃開始」を選択した際、各種工作の成否判定とアプローチの決定を行う。

#### A: 火計の成否判定（「火計準備」完了時のみ）
通常攻撃・夜襲攻撃問わず、「火計準備」に成功していた場合は接敵時に判定を行う。
- **成否の算定**: 
    - `火計成功率 = 50% + (火計計画担当者の知略 - 敵の抵抗値)`
- **成功効果**: 実交戦開始から**最初の5ターンの間、敵全部隊の戦闘力-10%**。

#### B: 全体作戦計画の成否判定（完了時のみ）
通常攻撃・夜襲攻撃問わず、「全体作戦計画」が完了している場合に接敵時に判定を行う。
- **不実行時の処理**: **誰も計画を立てていない場合、判定自体が発生せず、バフ・デバフともになし。**
- **成否の算定**: 
    - `成功率 = 50% + (全体作戦計画担当者の知略 - 敵の抵抗値)`
- **成功効果**: 実交戦開始から**最初の5ターンの間、自軍全部隊の戦闘力+10%**。
- **失敗の影響**: 計画がすり合わず混乱し、**全ユニットの疲労度+10、士気-5の状態**で戦闘開始。

#### C: 戦闘前鼓舞演説の成否判定（実行時のみ）
準備フェーズで「戦闘前鼓舞演説」を完了している場合、攻撃開始時に判定を行う。
- **実行制限**: **攻撃開始の直前1ターンのみ実行可能**。担当者をアサインした時点で他の作戦変更やターン進行がロックされ、最終決断（攻撃・撤退）に移行する。
- **成否の算定**: 
    - `成功率 = 指揮官の統率 + 25` (%)
- **成功効果**: **全部隊の士気+5**、および接敵から**3ターンの間攻撃力+5**。
- **失敗の影響**: 演説が空回りし、**士気-5、疲労+10の状態**で戦闘開始。

#### D: アタック分岐
- **通常攻撃**: 特殊効果なしで接敵。
- **夜襲攻撃**: **「夜襲準備アクション」完了済**の場合に選択可能な接敵オプション。
    - **知略の参照**: **「夜襲計画」を担当した者の知略**を「自身知略」として計算に使用する。
    - **成否判定**: 準備が完了していても、実行時の**「判定」**によって成否が分かれる。
    - **成功率の算定**: 
        - `夜襲成功率 = 50% + (実行者の知略 - 敵の夜襲抵抗値)`
        - `敵の夜襲抵抗値 = (敵統率者の統率 + 敵統率者の知略) / 2 + (敵部隊の平均士気 / 5)`
    - **シミュレーション**: 格下(知略30 vs 抵抗33)＝**47%** / 有能(知略80 vs 抵抗96)＝**34%**
    - **成功効果**: 敵士気-20、先制攻撃（自軍1ターン一方的攻撃）。
    - **失敗影響**: 自軍士気低下（-10）、敵に先制される（奇襲の失敗）。

## 3. 追撃・討ち取りシステム（盗賊討伐専用）

### 3-1. コンセプト

合戦（軍と軍）と異なり、盗賊退治の目的は「根絶・殲滅」であるため、敗走する敵に対しても徹底的な追撃を行う。
このロジックは「部隊全体の成果」を算出した後、その中から「個人の武功」を配分する2段階方式をとる。

### 3-2. STEP 1：部隊全体での合計討ち取り数を算出

逃走する敵に対し、部隊としてどれだけ効率的に追撃できたかを計算する。

#### 全体効率の計算
```typescript
合計追撃効率 = 基礎(50%) + 各種補正
clamp(10% 〜 90%)
```

- **疲労度補正**: `+(敵平均疲労 - 味方平均疲労) × 0.5`
- **機動力補正**: `+(味方の平均乗馬率 - 敵の平均乗馬率) × 30`
- **地形調査（退路遮断）**: `+25%` (準備アクションで実施済みの場合)
- **夜襲成功ボーナス**: `+15%`

#### 総討ち取り数
```typescript
総討ち取り数 = 逃走した敵の数 × 合計追撃効率
```

### 3-3. STEP 2：主人公（個人）の討ち取り数を分配

部隊の成果をメンバーで分け合う。味方の人数が少ないほど、1人あたりのチャンスが増える。

#### 計算式
```typescript
主人公の期待討ち取り数 = (総討ち取り数 / 味方の総人数) × 個人のレバレッジ

主人公討ち取り数 = random(0, 期待値 × 1.5) // まれに期待値を大きく上回る
```

#### 3-2. 個別配分ロジック (個人手柄の確定)

全体討ち取り数を、部隊内の各ユニットに以下の優先順位と重みで分配する。

1. **トドメ（マーク持ち）の優先回収**:
   - 交戦フェーズで主人公が「マーク」を付けた敗走中の敵への追撃は、**主人公の個人追撃効率に+30%のボーナス**がかかり、優先的に主人公の討ち取りとして計上される。
2. **機動力（馬）補正**: 
   - 馬上衆（馬あり）は歩兵よりも高い分配率を持つ。
3. **武芸ステータス**: 
   - 個人の武勇が高いほど、多くの敵を討ち取る。
4. **役割・配置**: 
   - 追撃隊としての役割、または前線にいた時間。

## 3. 立場別の功績（評価）フィロソフィー

準備アクションと討ち取り数の評価軸は、役職によって以下のように切り替わる。

- **総大将（指揮官）**: 
    - 準備は「敵を確実に殲滅するためのコスト」であり、準備自体の成功は加点されない。
    - 最終的な殲滅数（結果）がすべて。
- **構成員（部下）**: 
    - 上司から命じられた準備アクションの完遂は、独立した「功績（Merit）」として高く評価される。
    - 個人の武勇（討ち取り）と忠実な働き（準備成功）の両面で番付が決まる。

## 4. 状況変化ロジック（反作用）

「同じ手段ばかり繰り返す」「特定の管理を放置する」ことへのペナルティとして状況を変動させる。

### 4-1. 治安回復の反作用
- **平穏度の自然減衰**: 治安維持を怠ると、毎ターン固定値（例：-1）で平穏度が下がる。
- **盗賊の再編・強化**: 放置された盗賊カードは、一定ターンごとにランクが上がる（D→C等）。

### 4-2. 軍備増強の反作用
- **装備の劣化**: 装備整備を怠ると、訓練効率が低下する。
- **士気の低下**: 訓練を行わない期間が続くと、士気が徐々に低下する。

### 4-3. 計略の反作用
- **警戒度の上昇**: 同じ地域で計略を連続実行すると、警戒度が上がり、成功率が低下する。
- **警戒度の減衰**: 計略を行わない期間が続くと、警戒度は徐々に下がる。
