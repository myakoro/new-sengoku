## 修正履歴

### Version 1.1での主要な修正点

1. **経済バランスの大幅修正**
   - 馬上衆：借金レベルの厳しさ
   - 小頭：まだ貧乏、余裕なし
   - 組頭：ようやく余裕が出始める

2. **石高・兵糧・借金・軍役・恩賞の連動システム**
   - 知行地から年貢米
   - 扶持米として部下に支給
   - 合戦は自前の米で出陣
   - 軍役：知行地×3%（農兵＋徒士＋馬上衆）

3. **扶持米体系の確定**
   - 従僕：月0.3石
   - 徒士：月1.8石
   - 馬上衆：月3.5石
   - 小頭：月5石
   - 組頭以降：知行地のみ

4. **役職順序の修正**
   - 正：侍大将→部将（部将が上位）
   - 侍大将〜部将は「中盤」
   - 終盤：家老、城主、国攻略

5. **世界観の修正**
   - 史実ではなく「毎回創造される世界」
   - 地図・人物がプレイごとに変わる

## 2. 核心体験（何が面白いか）

### 2-1. 出世の加速度を体感する

```
徒士期（序盤）
├─ 何もない
├─ ギリギリの経済
├─ リスクを取るしかない緊張感
└─ 「次の出世までは耐えよう...」

↓ 馬上衆（地獄の入口）

馬上衆期
├─ 徒士を雇える（経費増）
├─ 馬を買える（大出費）
├─ 戦闘力は上がる（功績稼ぎやすい）
├─ でも借金レベルの貧乏
└─ 「出世しないと破綻する...」

↓ 必死の出世

小頭（やっと人間扱い）
├─ 給与が増える
├─ でもまだ貧乏
├─ 従僕・徒士を養う負担
├─ まだ足軽が身近にいる身分
└─ 「まだまだ余裕はない...」

↓ 徐々に改善

組頭（ブレイクポイント）
├─ 知行地がもらえる
├─ ようやく余裕が出始める
├─ 自家兵が増える
├─ 戦場での存在感
└─ 「やっと楽になってきた」

↓ 加速開始

足軽大将（加速期）
├─ 知行地が大幅増
├─ 経済的余裕
├─ 家臣団の充実
├─ 戦場での指揮権拡大
└─ 「止まれない」興奮

↓ 中盤へ

侍大将・部将（中盤）
├─ 一人前の武将
├─ 家の発展本格化
├─ 戦場の要
└─ 「ここからが本番」

↓ 終盤へ

家老・城主・国攻略（終盤）
├─ 戦国の中心プレイヤー
├─ 一国を任される
├─ 世代交代
└─ 「頂点に立った」達成感
```

**重要なポイント：**
- 馬上衆〜小頭：まだ苦しい（加速前）
- 組頭：ブレイクポイント（余裕の開始）
- 侍大将〜部将：中盤（まだ上がある）
- 家老以降：終盤（真の頂点へ）

### 2-2. 石高・兵糧・借金・軍役・恩賞の連動

本作の経済システムは**米（石高）を中心に全てが連動**する。

#### 経済の基本構造

```
知行地（石高）
  ↓
年貢米（税率60%）
  ↓
├─ 扶持米（部下への給与）
├─ 兵糧備蓄（合戦用）
└─ 余剰米の売却（現金化）
```

#### 役職ごとの経済感覚

- **徒士**：月0.75石の余剰（ギリギリ）
- **馬上衆**：月0.65石の余剰 + 借金40貫（地獄）
- **小頭**：月0.65石の余剰（まだ苦しい）
- **組頭**：月7.45石の余剰（ブレイク！）
- **足軽大将**：月8.25石の余剰（加速）
- **侍大将**：月38.85石の余剰（余裕）

**ポイント：**
- 馬上衆は借金返済に5年以上
- 小頭でも余剰はほぼ変わらず
- 組頭で急に楽になる（ブレイクポイント）
- 以降は指数関数的に余剰が増える

**詳細な数値とシステムは「9. 経済システム詳細」を参照**

### 2-3. 軍役システムの緊張感

**軍役 = 知行地 × 3%**（農兵＋徒士＋馬上衆で構成）

組頭の例：
- 軍役9人に対して、徒士2名＋馬上衆1名しか雇えない
- 残り6人は農兵で補う
- でも戦場で活躍するには武士を増やしたい
- 増やすと扶持米が増える→余剰米が減る

**ジレンマ：戦力 vs 経済**

詳細は「9-2. 軍役システム」を参照

### 2-4. 「何もない」から始める緊張感

- 従僕3人（名前なし、数値のみ）
- 知行地なし
- 貯金わずか
- 指揮権なし
- 「盗賊討伐に失敗したら死ぬかもしれない」プレッシャー
- **だからこそ、出世した時の解放感が大きい**

### 2-5. 「家」として生きる実感

- 個人ではなく「家」単位
- 従僕→家臣→家臣団の拡大
- 知行地の発展（組頭以降）
- 婚姻で家が繋がる（Version 1.0以降）
- 世代交代で歴史が続く（Version 1.0以降）
- **「家を大きくする」のが最終目標**

## 3. ターゲットプレイヤー

### 3-1. メインターゲット

**「歴史シミュレーション好き × システム理解力が高い × 長期プレイ可能 × 苦しさを楽しめる」**

具体的には：
- 信長の野望シリーズのファン
- Crusader Kings系の「家プレイ」が好き
- 太閤立志伝の「個人視点」が好き
- パラメータ管理が苦にならない
- **「最初は苦しいが徐々に楽になる」が好き**
- **「借金返済」「やりくり」が楽しい**
- 100時間プレイも厭わない

### 3-2. 年齢層

**25〜45歳（社会人）**

理由：
- 戦国時代への理解
- 長時間プレイの余裕
- システムの複雑さを楽しめる
- **「出世」「借金返済」「家計のやりくり」に共感しやすい**
- 「苦しい時期」を経験している

### 3-3. プレイスタイル

- **じっくり型**（一つ一つ考える）
- **計画型**（長期目標を立てる）
- **達成感重視**（数値の成長を楽しむ）

### 3-4. 求めているもの

- ✓ 他にない独自性
- ✓ リアルな戦国社会
- ✓ 成長の実感
- ✓ やり込み要素
- ✗ 爽快感（アクション）
- ✗ 手軽さ（カジュアル）

## 4. コアループ（短期・中期・長期）

### 4-1. 短期ループ（1週間〜1ヶ月）

```
【週次】
ターン開始
  ↓
状況確認（功績、米、借金、ライバル）
  ↓
公務選択（訓練 / 盗賊討伐 / 巡察）
  ↓
結果確認（功績、米、リスク）
  ↓
次週へ

【月次（4週ごと）】
評定
  ↓
出世判定（自分 & ライバル）
  ↓
扶持米支給（徒士〜小頭）or 年貢米収入（組頭以降）
  ↓
支出処理（従僕・徒士・馬上衆・借金返済）
  ↓
米の使い道選択（組頭以降：売却/備蓄/投資）
  ↓
次月へ
```

**ポイント：**
- 毎週「選択」する楽しさ
- 月次で「金のやりくり」を実感
- 借金返済の進捗が見える
- ライバルとの競争が緊張感を生む

### 4-2. 中期ループ（3ヶ月〜1年）

```
役職目標の設定
  ↓
功績の蓄積（公務の繰り返し）
  ↓
経済の安定化（借金返済 → 自家兵増強）
  ↓
出世達成
  ↓
新しい扶持米 or 知行地の獲得
  ↓
新しい軍役義務（組頭以降）
  ↓
次の役職を目指す
```

**ポイント：**
- 「次の出世まであと○○功績」が見える
- 借金返済の達成感
- 組頭で知行地獲得の喜び
- 出世後の収入増加が報酬

### 4-3. 長期ループ（3年〜10年〜30年〜）

```
【Version 0.1】徒士→馬上衆→小頭（3年）
├─ 出世の喜びを体験
├─ 借金地獄を体験
├─ 経済やりくりを理解
└─ ゲーム終了（完結）

【Version 0.5】小頭→組頭→足軽大将（5年）
├─ 知行地システム導入
├─ 軍役システム導入
├─ 家臣団の運営
├─ 戦闘での指揮
└─ 経済的余裕の獲得

【Version 1.0】足軽大将→侍大将→部将（10年）
├─ 経済の拡大
├─ 婚姻・引き抜き
├─ 世代交代
└─ 「中盤」の完成

【Version 2.0】部将→家老→城主→国攻略（30年）
├─ 戦略レイヤー
├─ 一国を任される
├─ 天下への関与
└─ 究極の達成感
```

## 5. プレイヤーの感情曲線

### 5-1. 徒士期（序盤）：苦しい → 耐える → 期待

```
感情の高さ
     ^
高い |
     |                    
     |                    
中程度|  ●------------●----●
     | /              \  /
     |/                \/
低い +------------------------> 時間
     開始  1年   2年  出世直前
```

**心理状態：**
- 開始：「何もない...」絶望
- 1年：「功績が貯まってきた」希望
- 2年：「まだ足りない...」焦燥
- 出世直前：「もうすぐ！」期待

**行動：**
- リスク回避（訓練中心）
- たまに盗賊討伐（ギャンブル）
- ライバルを気にする
- 貯金を貯める（馬上衆昇格後の借金を少しでも減らす）

### 5-2. 馬上衆期（地獄期）：達成 → 絶望 → 必死

```
感情の高さ
     ^
高い |  ●
     |  |＼
     |  | ＼
中程度|  |  ●-------●
     |  |  |
     |  |  |
低い +--●--+--------------> 時間
     出世 借金 3ヶ月 6ヶ月
        発覚
```

**心理状態：**
- 出世直後：「やった！」達成感
- 借金発覚：「40貫の借金...5年返済」絶望
- 3ヶ月：「返済がきつい...」苦しみ
- 6ヶ月：「出世しないと破綻する」焦燥

**行動：**
- 徒士を雇う（必須）
- 馬を買う（必須）
- 借金を抱える（避けられない）
- 盗賊討伐を増やす（功績稼ぎ）
- 「小頭になれば楽になる」と信じる

### 5-3. 小頭期（やっと人間）：解放 → でもまだ苦しい

```
感情の高さ
     ^

     |   /   ＼

     | /
     |●
低い +------------------------> 時間

```

**心理状態：**

- 3ヶ月：「でもまだ苦しい...」現実
- 6ヶ月：「借金返済が終わらない」悩み
- 1年：「組頭になれば楽になる」希望

**行動：**

**心理状態：**

## 6. 世界観とテーマ

### 6-1. 世界観の核心

**「毎回創造される、戦国に似た世界」**

- プレイごとに地図が変わる
- プレイごとに人物が変わる
- プレイごとに勢力が変わる
- でも社会構造は同じ（役職、家格、公務/私事）
- リプレイ性が高い

**社会構造：**
- 役職が権限を決める
- 家格が扱いを決める
- 公務と私事の分離
- 家単位の社会
- 実力主義の出世
- リアルな人間関係

**ポイント：**
- 史実ではない
- でも「戦国らしさ」は維持
- 秩序ある社会
- 実力主義の出世

### 6-2. テーマ

**「立身出世」**

- 身一つから始まる
- 実力で階段を登る
- 借金も返す
- 家を大きくする
- 世代を超えて続く
- 戦国を生き抜く

**副テーマ：**
- 苦しさからの解放（借金返済、経済安定）
- 加速度（組頭以降の爽快感）
- 秩序（社会の中で生きる）
- 継承（世代を超える）

### 6-3. 時代背景

- 戦国に似た架空の世界
- 大名領国制の形成期
- 家臣団構造の複雑化
- 貨幣経済と米経済の並存
- 実力主義の時代

**避けるもの：**
- 特定の大名・事件の再現
- 歴史の改変
- ファンタジー要素

```
信長の野望：
- 大名視点

- 個人（家）視点
```

```
太閤立志伝：
- 個人視点
- 自由度が高い
- カードシステム
```

```
CK：
- 欧州中世
- 複雑な継承システム
- 外交・陰謀中心
- 封建制

本作：
- 戦国日本
- シンプルな世代交代
- 出世・軍事中心
- 家臣団制
```

## 9. 経済システム詳細

### 9-1. 基本数値

```
【主人公がもらう扶持米（給与）】
- 徒士：年21.6石（月1.8石）
- 馬上衆：年42石（月3.5石）
- 小頭：年60石（月5石）
- 組頭以降：知行地のみ

【主人公が部下に支払う扶持米】
- 従僕：月0.3石/人
- 徒士：月1.8石/人
```

### 9-3. 軍役システム

```
【軍役計算】
軍役人数 = 知行地 × 3%

【軍役を満たす構成】
軍役人数 = 農兵 + 徒士 + 馬上衆

※小頭以上は軍役人数に含めない（指揮官だから）
※農兵は知行地の村から自動動員（コスト不要）
```

**例：組頭（知行地300石）**
```
軍役：9人

最小構成（コスト重視）：
- 農兵：9人
- 徒士：0名
- 馬上衆：0名
→ 扶持米：0石（農兵のみ）
→ 戦闘力：低い

バランス構成：
- 農兵：6人
- 徒士：2名
- 馬上衆：1名
→ 扶持米：7.1石/月
→ 戦闘力：中

戦力重視構成：
- 農兵：3人
- 徒士：3名
- 馬上衆：3名
→ 扶持米：15.9石/月
→ 戦闘力：高
→ 余剰米が赤字になる
```

**ジレンマ：**
- 農兵を多くする→コスト安い→戦闘力低い
- 武士を多くする→コスト高い→戦闘力高い→余剰米が減る

### 9-4. 借金システム

```
【借金発生タイミング】
- 馬上衆昇格時（ほぼ必須）
  └─ 馬購入：30貫
  └─ 徒士雇用一時金：10貫
  合計：40貫

- 任意借金（全役職）
  └─ プレイヤーが選択して借りる
  └─ 米不足、合戦準備、投資など

【返済】
- 毎月の余剰から任意額を返済
- 利子：なし（Version 0.1）
- 返済計画はプレイヤーが決める

【借金の上限】
- 徒士：10貫まで
- 馬上衆：50貫まで
- 小頭：100貫まで
- 組頭以降：無制限
```

### 9-5. 合戦時の兵糧

```
【兵糧消費】
- 兵1人：1日0.1石
- 兵1人：月3石（30日）

【合戦期間】
- 通常：10日間
- 大規模：20日間

【兵糧の調達】
原則：知行取り（組頭以上）は自前の米で出陣

選択肢：
1. 事前備蓄
   - 余剰米を売らずに倉庫に備蓄
   - 合戦時に使用
   - メリット：借金不要
   - デメリット：現金が手に入らない

2. 借金して購入
   - 合戦直前に米を借金で購入
   - メリット：普段は現金化できる
   - デメリット：借金が増える

3. 上司から借用
   - 上司に米を借りる
   - メリット：借金より安い
   - デメリット：恩を作る（貸し借り）
```

**例：組頭の合戦（軍役9人）**
```
必要兵糧：9人×10日×0.1石 = 9石
月の余剰米：7.45石
不足：1.55石

選択肢：
- 前月から1.55石を備蓄しておく
- 1.55石分（1.55貫）を借金
- 上司から借りる
```

### 9-6. 恩賞システム

```
【合戦後の恩賞】
功績に応じて以下から選択（or 組み合わせ）：

1. 加増（知行地増加）
   - 最も価値が高い
   - 永続的な収入増
   - 例：50石加増、100石加増

2. 米の恩賞
   - 一時的だが即戦力
   - 備蓄補充に使える
   - 例：50石、100石

3. 現金の恩賞
   - 借金返済・投資に使える
   - 例：10貫、20貫

4. 昇進
   - 役職が上がる
   - 扶持米が増える
```

## 10. GDDのまとめ

### 10-1. このゲームの本質

**「身一つから始めて、借金を抱え、苦しみながら戦国の階段を登る。組頭で世界が変わり、加速していく。家老になれば一国を任される。」**

### 10-2. 核心の核心

**「苦しさからの解放」と「出世の加速度」**

- 徒士期：苦しい
- 馬上衆期：もっと苦しい（借金地獄）
- 小頭期：まだ苦しい（でも希望）
- 組頭期：**ブレイクポイント（解放）**
- 以降：指数関数的拡大（止まらない）

### 10-3. プレイヤーが感じるもの

- ✓ 最初の苦しさ
- ✓ 馬上衆の絶望
- ✓ 小頭の希望
- ✓ 組頭の解放
- ✓ 加速の実感
- ✓ 家の発展
- ✓ 世代の継続（Version 1.0以降）
- ✓ 戦国を生きた実感

### 10-4. 他にないもの

- 石高・兵糧・借金・軍役・恩賞の完全連動
- 借金地獄（馬上衆）
- 軍役システム（知行地×3%）
- 組頭での劇的な変化
- 長い中盤（侍大将〜部将）
- 家老以降の終盤（一国を任される）
- プロシージャル生成の世界
