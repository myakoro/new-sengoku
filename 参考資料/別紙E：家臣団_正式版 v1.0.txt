■ E-1. 家臣団の基本定義

家臣団とは、
主家に属し、役職・知行・軍事任務・公務を担う構造的組織 である。

本作の家臣団は以下で構成される：

主家直臣

外様家臣（召抱え）

家内奉公人（若党／旧称：従僕）

軍事階層（徒士・馬上衆・小頭・組頭・大将 ほか）

家臣団は 家格（別紙B）・役職（別紙C）・公務（別紙D）・軍事（別紙G/L） の連動システムで管理される。

■ E-2. 家臣構造（階層と役割）

家臣団は以下の階層で構成される：

階層	主な構成員	説明
上級家臣	部将・侍大将	軍団級指揮・主要知行
中級家臣	足軽大将・組頭	軍事の中核（100〜300）
下級家臣	小頭・馬上衆・徒士	小隊長・精鋭・歩兵
家内奉公	若党	家内仕事＋護衛

v0.9 では 軍事階層の刷新 があり、
特に「馬上衆」を新設し、家臣団の層が精密化された。

■ E-3. 若党・徒士・馬上衆の定義（v0.9以降の基礎）

本作では 若党 → 徒士 → 馬上衆 → 小頭 → 組頭 → 足軽大将 が正規昇格ルートである。

● E-3-1. 若党（家内奉公＋戦場従者）

若党は、主人（徒士・馬上衆・小頭など）の

- 身の回りの世話
- 家内雑務
- 戦場同行・近接戦闘

を担う従者である。身分的には足軽と同格だが、ゲーム上は以下の特徴を持つ。

- 主人公などが **個人名で雇う家臣** として扱う
- 武芸・統率・知略・政務の4能力を持つ
  - 能力合計は概ね 200〜270 の範囲でばらつく
  - ただし、主人公がゲーム開始時に雇う初期若党だけは、特例として 230〜280 程度の高水準から生成される
- 戦場では主人の直近に配置され、小規模戦闘では実戦戦力として数えられる
- 主人の任命だけで「徒士」へ昇格できる（追加条件なしの裁量昇格）

他家の若党は、人口全体の能力分布に基づいて生成される。プレイヤー以外の家中における名もなき従者は、ゲーム上は「足軽」としてまとめて扱ってよい。

● E-3-2. 徒士（基礎武士）

小隊（25人）を構成する標準的な武士。

- 公務も可能
- 騎乗不可（徒歩戦）
- 個人武力は小隊戦力に加算
- 死亡時：小隊士気 −2〜−4

徒士は原則「本人1名＋若党（最大2名）」という小さな単位で行動する。若党は足軽同身分だが、名あり・4能力持ちで将来徒士へ昇格し得る“準人材”として扱われる。

● E-3-3. 馬上衆（上位徒士・騎馬武者）

徒士から昇格した「騎乗可能な上位兵階層」。

● 特徴

- 徒士より高い個人戦闘力を持つことが多い（ただし例外あり）
- **本人＋部下7人＝8人単位** の小隊を率いる
  - 部下7人は、徒士・若党・足軽が混在し得る
- 騎乗可能（馬の保有は自前）
- 指揮権は自隊8人に限られる（小頭の下で動く）

● 戦場での扱い

- 自隊8人を率いる前衛・突撃役
- 馬保有者は馬の質に応じた戦闘力加算（例：武芸に +30〜50 相当）
- 死亡時：小隊士気 −2〜−5（個人枠の喪失として扱う）

● 昇格

- 徒士上位 → 馬上衆
- 馬上衆 → 小頭

■ E-4. 小頭・組頭・足軽大将・侍大将の位置づけ（軍事階層）

v0.9以降、隊階層と軍事役職は以下のように運用する。

● E-4-1. 小頭（25人小隊長）

馬上衆からの昇格。

- 指揮対象：25人小隊
- 編成内訳：
  - 馬上衆部隊 8人 ×2隊 ＝16人
  - 小頭直属部隊 9人（小頭本人＋直属8人）

- 個人武勇の影響が最も大きい階層
- 指揮上限：25人
- 騎乗可能

戦場での責務：

- 小隊の生存（損耗管理）
- 小隊戦力の直接行使
- 追撃・崩壊時の被害状況の抑制

● E-4-2. 組頭（100人中隊長）

小頭からの昇格。

- 指揮対象：100人中隊
- 編成内訳：
  - 小頭小隊（25人）×3 ＝75人
  - 組頭直属 25人
    - 組頭本人
    - 直属の小隊運用要員（徒士・若党・足軽・馬上衆など）

- 指揮上限：100人
- 騎乗可能

戦場での責務：

- 小隊ローテーション
- 戦線の厚み調整
- 中隊単位の生残判断

● E-4-3. 足軽大将（300人大隊長）

組頭からの昇格。

- 指揮対象：300人大隊
- 編成内訳：
  - 組頭中隊（100人）×3 ＝300人
  - このうち1中隊は足軽大将直属中隊として運用可能

- 指揮上限：300人
- 騎乗可能

戦場での責務：

- 中隊のローテ・後退
- 敵の崩壊判断
- 追撃強度の決定
- 戦場全体の生存最適化

● E-4-4. 侍大将（900人規模）

足軽大将からの昇格。

- 指揮対象：900人規模の軍団
- 編成内訳：
  - 足軽大将大隊（300人）×3 ＝900人
  - このうち1大隊は侍大将直属大隊として運用可能

侍大将は、戦場全体の戦略方針と政治的責任を負う上級家臣として位置づけられる。

■ E-5. 家臣と知行地（密接な関係）

本作では、
家臣は知行地を持ち、そこから兵を出す。

● 人口と兵の関係（v0.9更新）

兵1人ごとに 出身知行地ID を保有

死亡・廃兵・逃散 → 出身地の人口が −1

負傷は人口に影響なし

→ したがって、
大量の借り兵を失うと “貸してくれた他家の人口” が減るため、評価に影響（別紙J）。

■ E-6. 家臣の評価（戦功と忠誠）
● v0.9では評価入力が変更

家臣の評価には：

戦功（士気ダメージ／崩壊／追撃／損耗効率）

借り兵の損耗

忠誠／貸し借り

公務実績

が関わる。

評価モデル自体は別紙Jだが、
v0.9により 戦場データの入力項目が増加 した。

■ E-7. 家臣団の編成（軍事視点）

本作の家臣団は軍事階層と密接に関係する。

E-7-1. 小隊（25人）編成

小隊（25人）は以下のように構成される：

- 馬上衆部隊：8人 ×2隊 ＝16人
  - 各隊：馬上衆本人1＋部下7（徒士／若党／足軽）
- 小頭直属部隊：9人
  - 小頭本人＋直属8人

小隊は「個々の戦闘力の集合体」であり、

- 徒士・若党・足軽の質
- 馬上衆・小頭の武芸と統率
- 保有馬の質

が小隊全体の強さを左右する。

E-7-2. 中隊（100人）編成

小隊4つ＋予備で構成。

組頭が次を管理：

小隊の疲労

士気

ローテーション

生存率最適化

E-7-3. 大隊（300人）編成

中隊3つ＋予備で構成。

足軽大将が戦術判断すべてを担当：

後退・撤退

戦線の再配置

追撃判断

■ E-8. 家臣の出世（階層上昇）

出世ルートは以下の通り：

徒士 → 馬上衆 → 小頭 → 組頭 → 足軽大将 → 部将 → 侍大将

出世要件は別紙I：

年間戦功

公務

忠誠

貸し借り

能力値（最低ライン）

馬上衆の導入により、
徒士上位 → 馬上衆 → 小頭 の流れが自然になった。

■ E-9. 家臣団における若党の扱い（v0.9安定版）

若党は家臣団内で特殊階層。

主人の戦闘補助

個人護衛

小隊戦力不算入

死亡は本人のみ微ペナルティ

知行地人口にも影響なし

将来的に 1.1 で：

若党の質

名門家の若党の優秀さ

主人との忠誠関係

などを拡張する。

■ E-10. v0.9 スコープ外（将来拡張）

馬の詳細ステータス（質・速度・馬体）

兵科（槍・弓・鉄砲）の正式分類

家臣団同士の派閥

軍団規模の複数大隊統合

家臣の生活コスト・扶持の経済処理

若党の階層成長

馬上衆の一家制度イベント
