# 戦国立身出世SLG v0.15 BGM 音源生成プロンプト（アンチグラビティ用）

## 0. 目的
戦国時代を舞台にした「渋く・硬派」なSLGの世界観に合うBGMを、
**通常 / 戦闘 / 評定**の3種類で用意する。

## 1. 格納先（この通りに置くとゲームで自動再生される）
生成した音源は、リポジトリの以下へ配置する。

- 通常BGM: `public/bgm/main.(mp3|ogg|wav)`
- 戦闘BGM: `public/bgm/battle.(mp3|ogg|wav)`
- 評定BGM: `public/bgm/council.(mp3|ogg|wav)`

推奨は `mp3`（可能なら 320kbps、難しければ 192kbps以上）。

## 2. 共通条件（全曲共通）
以下の制約を必ず満たすBGMを作ってください。

```text
戦国時代（和風・渋い・質実剛健）のゲームBGMを作ってください。
現代的なEDM/シンセの主張、ポップなコード進行、明るすぎるメロディは避ける。
主な楽器は和楽器寄り（尺八・篠笛・琴・三味線・太鼓）＋控えめな弦/低音（ヴィオラ/チェロ相当の持続音）で良い。
旋律はペンタトニック寄り、短調/陰影を重視。過度にホラーにはしない。

ループ前提のBGMにする：
- 冒頭からすぐ雰囲気が出る（長い導入なし）
- 10〜20秒の短い曲ではなく、最低でも60〜120秒程度のループ素材
- ループの継ぎ目が自然（終止感を作りすぎない）
- 1曲の中で微妙な展開（A→A'→B→A等）を入れて単調さを避ける

音質/出力：
- ステレオ
- 44.1kHz
- 16bit 以上
- クリップ/歪みなし
- 最終出力は mp3（可能なら320kbps、難しければ192kbps以上）
```

## 3. 曲別プロンプト

### 3-1. 通常BGM（Main）
用途：メイン画面、買い物、編成など「日常/内政」
狙い：落ち着き・渋さ・少しの緊張感（戦の予感）

```text
[曲名] 「城下の気配」

上の共通条件に加えて：
- テンポ：80〜100 BPM（ゆっくりめ）
- 雰囲気：静かな城下、夜明け前、油断はできないが落ち着いている
- リード：尺八または篠笛の短いフレーズを控えめに反復
- リズム：太鼓は軽く、目立ちすぎない（拍を支える程度）
- 低音：地面の重さを感じる持続音（薄いドローン）を入れる
- キラキラした明るい音色は禁止、現代的なドラムセットは禁止

納品ファイル名：main.mp3
```

### 3-2. 戦闘BGM（Battle）
用途：盗賊討伐の決戦・戦闘中
狙い：緊迫・疾走・太鼓の推進力（ただし現代感は抑える）

```text
[曲名] 「乱戦」

上の共通条件に加えて：
- テンポ：120〜150 BPM（速め）
- 雰囲気：小規模戦闘〜合戦前哨戦。勇ましいが英雄的すぎない（リアル寄り）
- リズム：和太鼓中心。裏拍や変化で焦燥感を出す（単純なドンドコ祭りにしない）
- リード：篠笛/尺八の短い切れ味あるフレーズ（叫びすぎない）
- 緊張感：不協和は少しだけ（やりすぎるとホラーになるのでNG）
- 勝利感の強いサビは禁止（ループさせても疲れにくい構造）

納品ファイル名：battle.mp3
```

### 3-3. 評定BGM（Council）
用途：小頭評定（会議/政治/圧）
狙い：重圧、格式、静かな威圧感（恐怖ではなく権威）

```text
[曲名] 「評定」

上の共通条件に加えて：
- テンポ：60〜80 BPM（重く遅め）
- 雰囲気：上役の前、緊張する会議、張り詰めた空気
- リズム：太鼓は極力抑えて、低音の脈動や間で圧を出す
- 旋律：琴/三味線の断片的なフレーズ（余韻を活かす）
- 低音：重いドローン、一定の不安感（ただしホラー化しない）
- 派手な盛り上がり禁止。終始「静かに怖い」寄りの威圧

納品ファイル名：council.mp3
```

## 4. 追加の一言（任意だが推奨）
対応している場合、各曲に以下も添える。

- ループ用に、末尾1〜2小節は自然に頭へ戻れるように調整して
- イントロで無音が長くならないように（開始0.5秒以内で鳴り始める）
