## 1. 概要

### 1-1. 設計方針

**Version 0.1の画面設計の核心：**
- **シンプル第一**：複雑なUIは避け、必要最小限の情報のみ表示
- **情報の優先順位**：プレイヤーが今知るべき情報を最優先で表示
- **選択の明確化**：何を選べるか、何が起こるかを明確に
- **フィードバックの即時性**：行動の結果をすぐに表示
- **緊張感の演出**：数値の変化、ライバルとの比較を視覚化

### 1-2. デザインコンセプト

```
【テーマ】
戦国時代の緊張感 × 現代的な見やすさ

【カラースキーム】
- ベース：ダークグレー（#2C2C2C）
- アクセント：金色（#D4AF37）
- 警告：赤（#C62828）
- 成功：緑（#2E7D32）
- テキスト：白（#FFFFFF）、薄灰色（#CCCCCC）

【フォント】
- 日本語：游ゴシック、メイリオ
- 数値：等幅フォント（Consolas、Monaco）
```

### 1-3. 技術スタック

```
フロントエンド：React + TypeScript
スタイリング：Tailwind CSS
状態管理：Zustand
ルーティング：React Router
アニメーション：Framer Motion
```

#### 要素

| 要素 | 説明 | 機能 |
|------|------|------|
| 才能スライダー | 各ステータスの上限を設定 | ドラッグで調整、合計300まで |
| 残りポイント表示 | 未配分のポイント | リアルタイム更新 |
| 現在値表示 | ゲーム開始時の実際の値 | 才能上限の80% |
| 従僕表示 | 従僕3名の武芸値 | 40〜60のランダム |
| 才能振り直しボタン | 主人公の才能をランダム再生成 | 何度でも可能 |
| 従僕振り直しボタン | 従僕の武芸をランダム再生成 | 何度でも可能 |
| 決定ボタン | キャラメイク完了 | メイン画面へ遷移 |

#### 従僕の初期武芸

- 従僕3名、全員武芸40〜60のランダム
- 「従僕を振り直し」ボタンで再抽選可能

#### バリデーション

- 合計が300を超えないこと
- 各ステータスは10〜100の範囲
- 決定ボタンは合計が300の時のみ有効

## 8. 週数表示の実装

### 8-1. 表示形式

**採用形式**: 「1575年3月2週目」

```javascript
const GAME_START_YEAR = 1575

function formatWeek(week) {
  // 年の計算
  const yearsPassed = Math.floor((week - 1) / 52)
  const year = GAME_START_YEAR + yearsPassed
  
  // 月の計算（1週=1週間、4週=1ヶ月）
  const weekInYear = ((week - 1) % 52) + 1
  const month = Math.floor((weekInYear - 1) / 4) + 1
  
  // 月内の週（1〜4週目）
  const weekInMonth = ((weekInYear - 1) % 4) + 1
  
  return `${year}年${month}月${weekInMonth}週目`
}

// 例：
// week = 1  → "1575年1月1週目"
// week = 4  → "1575年1月4週目"
// week = 5  → "1575年2月1週目"
// week = 52 → "1575年12月4週目"
// week = 53 → "1576年1月1週目"
// week = 89 → "1576年9月1週目"
```

### 8-2. 表示理由

1. **戦国時代の臨場感**
   - 実際の年号を使うことで時代感が増す
   - 1575年は長篠の戦いの年（戦国時代の象徴的な年）

2. **月と週の両方が明確**
   - 季節感が分かる（1月=冬、3月=春など）
   - 週単位の進行も把握しやすい

3. **固定ゴールを示さない**
   - 年数表示なので終わりが見えない
   - 小頭到達でエンディング

### 8-3. 月と週の対応表

| 週（年内） | 月 | 週（月内） |
|-----------|----|-----------|
| 1〜4週 | 1月 | 1〜4週目 |
| 5〜8週 | 2月 | 1〜4週目 |
| 9〜12週 | 3月 | 1〜4週目 |
| ... | ... | ... |
| 49〜52週 | 12月 | 1〜4週目 |

### 8-4. 開始日時

```
ゲーム開始: 1575年1月1週目
```

**1575年を選んだ理由**:
- 長篠の戦いの年（戦国時代の象徴的な年）
- 実際の年号を使うことで時代感が増す
- 季節感が分かる（1月=冬、3月=春など）
