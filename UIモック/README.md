## カラースキーム

| 色 | 用途 | カラーコード |
|----|------|-------------|
| ベース | 背景 | #2C2C2C |
| ダーク | 濃い背景 | #1a1a1a |
| アクセント | 重要情報、金色 | #D4AF37 |
| 成功 | 成功、増加 | #2E7D32 |
| 警告 | 失敗、減少 | #C62828 |
| テキスト | 通常テキスト | #FFFFFF |
| サブテキスト | 補助テキスト | #CCCCCC |
| 無効 | 無効状態 | #555555 |
