# アンチグラビティへの指示: UI・仕様の更新

## 概要

盗賊討伐画面（SC-08）およびメイン画面（SC-03）のUIと仕様が更新されました。以下の変更を動的実装に反映してください。

---

## 0. メイン画面の修正（重要）

### 0-1. 経済セクションに「金」を追加

現在の表示:
```
経済
米      0.63石
借金    0.00貫
```

修正後:
```
経済
米      0.63石
金      0.50貫  ← 追加
借金    0.00貫
```

### 0-2. 月次処理のタイミング変更

**旧仕様**: 3週目と4週目の間で月次処理を実行
**新仕様**: 4週目のあとに月次処理を実行

```
旧: 1週目 → 2週目 → 3週目 → [月次処理] → 4週目 → 1週目...
新: 1週目 → 2週目 → 3週目 → 4週目 → [月次処理] → 1週目...
```

月次処理の内容:
- 扶持米の支給
- 家臣への給与支払い
- 借金利息の計算
- 月次報告画面（SC-06）の表示

### 旧仕様
```

```

### 新仕様
```

進行: 第2週 / 4週（準備期間）[プログレスバー]
```

### 実装要件

- 期限は赤色（#FF6B6B）で強調
- プログレスバーで進行状況を視覚化
- フェーズ表示（「準備期間」または「最終週」）

## 2. 行動ログの追加

### 新仕様

各行動の結果を以下の形式でログ表示：

```typescript
interface ActionLogEntry {
  week: number | "開始"     // 行動した週
  actionName: string        // "偵察", "足軽雇用", "任務開始"
  result: "成功" | "失敗" | "―"
  detail: string            // 具体的な効果
}
```

### 表示例
```
[第2週] 偵察     [成功] 敵の詳細情報を入手。人数4名、戦闘力85と判明。
[第1週] 足軽雇用 [成功] 足軽5名を雇用。戦力+160。
[開始]  任務開始 [―]   盗賊討伐（中規模）を開始。期限: 4週間後。
```

### スタイル
- 成功: 緑色バッジ（背景#1B5E20、文字#81C784）
- 失敗: 赤色バッジ（背景#B71C1C、文字#EF9A9A）
- ログは新しい順（上が最新）

## 3. 戦闘力計算の変更

### 旧仕様
```javascript
// 従僕の戦力（武芸の平均 × 0.2 × 人数）
power += juubokuAverage * 0.2 * player.juuboku.length
```

### 新仕様
```javascript
// 個人戦闘時（盗賊討伐・護衛任務）は従僕100%
player.juuboku.forEach(j => {
  power += j.combat  // 0.2ではなく1.0
})
```

**注意**: 戦争時（Version 0.5以降）は従僕0.2のまま。盗賊討伐では100%。

## 7. ミッションフロー確認

```
盗賊討伐（4週間の場合）:
  第1週: 準備行動（偵察/偽情報/内応者買収/足軽雇用/待機）
  第2週: 準備行動
  第3週: 準備行動
  第4週: 最終週（通常攻撃/夜襲/諦める）← 戦闘で1週間消費

盗賊討伐（2週間の場合 = 小規模）:
  第1週: 準備行動
  第2週: 最終週（通常攻撃/夜襲/諦める）
```
