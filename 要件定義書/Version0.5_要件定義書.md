## 3. 知行地システム

### 3-1. 基本概念

```
【知行地とは】
- 大名から与えられた土地
- 年貢米を徴収できる
- 開発・投資で収益UP
- 軍役の義務が発生

【獲得タイミング】
- 組頭昇進時に初めて知行を得る
- 功績に応じて加増される
```

### 3-2. 知行地の特徴タグ

```
【立地特徴】
- 街道沿い → 行商が来やすい、商人親密度UP
- 山沿い → 木材・鉱物、防衛しやすい
- 川沿い → 水運、水田に適する
- 国境沿い → 危険だが手柄の機会、軍役多い
- 平野部 → 農業生産高い、防衛しにくい

【開発選択肢】
- 道を伸ばす → 街道接続、行商ルート開通
- 市を開く → 商人招致、収入UP
- 堤防を築く → 水害リスク減、収穫安定
- 砦を築く → 防衛力UP、軍役で有利
```

### 3-3. 年貢と収入

```
【年貢】
- 知行高の40〜60%が実収入
- 開発度で変動
- 災害で減少

【収入の使い道】
1. 備蓄 → 兵糧として保管
2. 売却 → 金に換える
3. 投資 → 開発に使う
4. 家臣扶持 → 配下への給与
```

## 4. 軍役システム

### 4-1. 基本概念

```
【軍役とは】
- 知行に応じた兵力動員義務
- 大名の合戦に参加
- 兵糧は自前で用意

【軍役の計算】
- 知行100石 → 兵5〜10名程度
- 役職・功績で変動
```

### 4-2. 合戦参加

```
【合戦の流れ】
1. 軍役召集（1〜2ヶ月前に通知）
2. 兵糧準備
3. 出陣
4. 戦闘（複数ターン）
5. 帰還・報告

【戦功評価】
- 敵士気ダメージ
- 崩壊・追撃
- 損耗効率
- 下知達成
```

---

## 5. 戦闘システム詳細

### 5-1. 戦闘フェーズ

```
【乱戦フェーズ】
- 方針選択（攻撃/平常/防御）
- 戦闘力比較
- 疲労・士気変動
- 損耗判定

【追撃フェーズ】
- 敵崩壊時に発生
- 馬があると手柄確率UP
- 大きな損耗を与える

【撤退フェーズ】
- 自軍崩壊時に発生
- 馬があると被害軽減
- 殿軍の選定
```

### 5-2. 馬の死亡

```
【死亡条件】
- 戦闘中：5%/ターン
- 撤退時：15%
- 追撃時：3%

【死亡時】
- 馬を失う
- 再購入が必要（8貫）
```

## 6. 配下システム

### 6-1. 配下の区分

```
【雇用（自前）】
- 自分で雇った配下
- 給与は自分持ち
- 死傷責任なし（功績ペナルティなし）

【借り（上司から）】
- 上司から借りた配下
- 給与は上司持ち
- 死傷責任あり（功績ペナルティ）
```

### 6-2. 借り配下の死傷責任

```
【死亡時】
- 功績-10（下知達成時）
- 功績-20（下知未達時）

【重傷時】
- 功績-3（下知達成時）
- 功績-6（下知未達時）

【返却】
- 任務終了時に返却
- 負傷状態で返却すると評価DOWN
```

## 7. 家臣団

### 7-1. 名前付き家臣

```
【徒士】
- 名前あり
- 4ステータス（武芸・統率・知略・政務）
- 成長あり
- 忠誠度あり

【馬上衆】
- 名前あり
- 4ステータス
- 馬の有無
- 成長あり
```

### 7-2. 家臣の成長

```
【経験値】
- 戦闘参加で獲得
- 公務参加で獲得

【成長】
- 経験値で能力UP
- 上限は素質で決まる
```

## 8. 兵糧システム

### 8-1. 兵糧消費

```
【消費量】
- 兵1人：1日0.1石
- 兵1人：月3石（30日）

【合戦期間】
- 通常：10日間
- 大規模：20日間
```

### 8-2. 兵糧の調達

```
【選択肢】
1. 事前備蓄
   - 余剰米を売らずに保管
   - メリット：借金不要
   - デメリット：現金が手に入らない

2. 借金して購入
   - 商人から借金
   - メリット：即座に調達可能
   - デメリット：金利負担
```
