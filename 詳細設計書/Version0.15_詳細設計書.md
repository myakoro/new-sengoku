## 1. この文書の位置づけ

本書は Version 0.1 の基礎仕様に対して、Version 0.15 で追加・変更された仕様差分をまとめる。

---

## 2. 追加仕様（Version 0.15）

### 2-1. 馬上衆昇進時の貸与足軽（無料）

- 対象：徒士 → 馬上衆への昇進
- 内容：上司から足軽が一定数「貸与」（無料）される
- 貸与足軽：**5名**
- これらは「雇用した徒士」とは別枠で管理され、**扶持米の支出計算には含めない**

### 2-2. 編成（7スロット）

馬上衆以上で、配下を 7 スロットに編成して管理する。

- 編成対象：若党 / 貸与足軽 / 徒士
- 戦闘参加者：原則として **編成に入っているユニットのみ**
  - 編成が空の場合は、利便性のため「若党（＋貸与足軽）」を自動で参加させてもよい

### 2-3. 戦闘外での家臣状態確認

戦闘中以外でも、配下の負傷状況を一覧で確認できる。

- 画面：メイン画面（Main）
- 導線：左サイドバー「家臣」パネルに `家臣の状態` ボタンを追加し、モーダルで一覧表示
- 表示内容：若党 / 徒士 / 貸与足軽 / 馬上衆
  - 能力：武芸 / 統率 / 知略 / 政務
    - 若党 / 馬上衆は 4能力を表示
    - 徒士 / 貸与足軽（足軽系）は将来的な自前雇用を含め、現仕様では「武芸」のみを保持するため、他の能力は `―` 表示
  - 状態：`正常` / `軽傷` / `重傷`
  - 回復まで：`injuryWeeksRemaining`（0なら `―`）
  - HP目安：負傷状態から算出する目安値（戦闘中のHPとは別）
    - 正常: 100
    - 軽傷: 80
    - 重傷: 0

### 2-4. BGM（通常/戦闘/評定）

画面に応じてBGMを自動再生・切替する。

- BGM種別
  - 通常: `main`
  - 戦闘: `battle`
  - 評定: `council`
- 切替ルール（画面ID）
  - `bandit-mission`: 戦闘BGM
  - `kochou-evaluation`: 評定BGM
  - `title` / `ending`: 無音（停止）
  - 上記以外: 通常BGM
- 音源配置
  - `public/bgm/main.(mp3|ogg|wav)`
  - `public/bgm/battle.(mp3|ogg|wav)`
  - `public/bgm/council.(mp3|ogg|wav)`
  - 音源が未配置の場合でもゲーム進行は止めない（無音で動作する）

### 2-5. 盗賊討伐（準備期間）: 様子を見る

盗賊討伐の準備期間（決戦前）に、計略を使わずにターンを消費して進行できる。

- 画面：盗賊討伐（BanditMission）
- 操作：`様子を見る`
- 効果：任務の週を進める（計略は使用しない）
